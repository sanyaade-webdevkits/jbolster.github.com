--- layout: post title: Random sort on IEnumerable object tags: - c# - code - dotnet - extension - linq - programming status: publish type: post published: true --- Quick extension on an IEnumerable object to return the collection in a random order: public static IEnumerable Random(this IEnumerable source) { return source.OrderBy(x => Guid.NewGuid()); } For example, to return a random set of 5 items from the collection: var randomSet = myEnumerable.Random().Take(5); If anyone knows of a better way, let me know :)
